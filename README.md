O projeto consiste na criação de uma Locadora de Vídeo Web para um trabalho do Ensino Técnico.  

Deve conter, necessariamente:  
1. As novas tags do HTML5 e CSS3  
2. Um nome e logotipo para a locadora  
3. Padrão de nomenclatura para as páginas HTML. Ex:cadatroUsuario.html  
4. Deverão haver dez filmes e uma trilha sonora para cada  
5. Três gêneros de filmes, a critério dos alunos  
6. O site deve ser hospedado em algum webhost
